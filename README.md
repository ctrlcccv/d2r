# Traderie D2R 최근 거래 가격 도우미

이 프로젝트는 Traderie 최근 거래 파일을 업로드해, 내 아이템의 변동 옵션과 수량 기준으로 비슷한 거래를 빠르게 비교하는 로컬 앱입니다.

## 중요 안내

반드시 본인 계정만 사용하고, Traderie의 이용약관과 요청 제한을 지켜야 합니다.
이 프로토타입은 로그인을 우회하지 않습니다. `.session/traderie`에 저장된 Playwright 브라우저 프로필에 사용자가 직접 한 번 로그인하는 방식입니다.

## 설치

```bash
npm install
```

## GitHub Pages 배포

이 프로젝트는 GitHub Pages로 올릴 수 있게 정적 배포 구성을 함께 포함하고 있습니다.

1. 이 저장소를 GitHub에 push
2. GitHub 저장소 `Settings > Pages`에서 `Source`를 `GitHub Actions`로 선택
3. 기본 브랜치를 `main`으로 사용하면, 이후 `main`에 push할 때마다 자동 배포

로컬에서 Pages용 정적 결과물을 직접 만들고 싶다면:

```bash
npm run build:pages
```

그러면 `dist/` 폴더가 생성되고, GitHub Pages에 올라갈 파일이 준비됩니다.

## 사용 방법

1. 앱 실행

```bash
npm run app
```

2. 브라우저에서 `http://localhost:4173` 열기

3. 페이지의 `스크립트 복사` 버튼을 눌러 수집 스크립트를 복사

4. Traderie에서 원하는 아이템의 `Recent Trades` 페이지를 로그인 상태로 연 뒤, 개발자도구 `Console`에 붙여넣고 실행

5. 다운로드된 `traderie-page-export.json` 파일을 앱 페이지에 업로드

6. 업로드하면 아이템 이름과 변동 옵션 입력칸이 자동으로 만들어집니다

7. 필요한 값만 수정한 뒤 `결과 보기` 클릭

## 예시 옵션 입력

```text
수량: 40
방어력 증가: 30
방어력: 87
```

## 보조 CLI 명령

기존 커맨드 방식도 그대로 남겨두었습니다.

```bash
npm run import -- ~/Downloads/traderie-page-export.json
npm run recommend
```

## 현재 한계

- 사이트가 동적 앱이라, 로그인 후 실제로 보이는 최근 거래 화면 구조에 따라 파싱 결과가 달라질 수 있습니다.
- 현재는 텍스트 기반 파서라서 모든 옵션과 거래 형식을 완벽하게 구조화하지는 못합니다.
- 가격 정규화는 현재 룬과 일부 재료 기준의 초기 버전입니다.
- GitHub Pages 배포본은 정적 사이트이므로, 브라우저에서 JSON 파일을 직접 업로드해 사용하는 방식입니다.
